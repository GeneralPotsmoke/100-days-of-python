#### Day 65: Web Design School - How to Create a Website that People will Love
**Challenge:** Design a visually appealing and user-friendly website. Focus on layout, typography, and color schemes.

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Web Design School</title>
    <link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/4.5.2/css/bootstrap.min.css">
    <style>
        body {
            font-family: 'Arial', sans-serif;
            background-color: #f8f9fa;
        }
        h1 {
            color: #343a40;
            text-align: center;
            margin-top: 20px;
        }
        p {
            color: #6c757d;
            text-align: center;
        }
        .container {
            margin-top: 50px;
        }
    </style>
</head>
<body>
    <div class="container">
        <h1>Welcome to Web Design School</h1>
        <p>Learn how to create beautiful and user-friendly websites.</p>
        <div class="row">
            <div class="col-md-4">
                <h2>Layout</h2>
                <p>Organize your content in a logical and visually appealing manner.</p>
            </div>
            <div class="col-md-4">
                <h2>Typography</h2>
                <p>Choose fonts that are easy to read and match the tone of your website.</p>
            </div>
            <div class="col-md-4">
                <h2>Color Scheme</h2>
                <p>Select colors that complement each other and enhance readability.</p>
            </div>
        </div>
    </div>
</body>
</html>
```
